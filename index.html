Travel Planner App

This is an app I made with a group of schoolmates in my analytics module. It was designed for tourists to plan, or randomize, their itinerary. The app was made with the "Shiny" package from the R language, which is used to build interactive web apps. Interestingly, it was my first encounter with the bootstrap framework - not a web development module!

My role was to come up with the interface and to integrate the APIs with the app. Through the project, I learnt bootstrap and how to integrate my app with multiple APIs (Google Maps API, Google Places API, Weather API)

Telegram Bots

This is an app developed for my volunteering organization when to keep track of guest impressions. It gives the hosts convenience in updating information about each guest, as they can do it via Telegram instead of opening the database to do so.

The app was developed as a web app, hosted on Google's server. It makes use of the Telegram API to build a friendly user interface and is connected to a Google Sheet as the database. Aside from this, I have created a few more Telegram Bots, such as a Coffee Subscription Bot, for my final year project's client.
